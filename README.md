## Description
An accounts management demo. The user has Admin rights and is able to create/edit/delete accounts with a name, email and permissions.

## TechUsed
# HTML,CSS,JS,Node.js

## Installation

1. Clone repo
2. run `npm install`

## Usage

1. run `node server.js`
2. Navigate to `localhost:8080`
